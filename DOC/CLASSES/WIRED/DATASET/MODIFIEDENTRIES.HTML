Class Wired::Dataset::ModifiedEntries
In: wired-dataset.rb
Parent: ChangedEntries

Represents all of the Registry entries found modified in a Registry difference. ModifiedEntries#write does all of the real work; just instantiate and use ModifiedEntries#write to output all of its records to the dataset file
