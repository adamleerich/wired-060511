wired-diff.rb

Synopsis

Classes to enable the differencing of Windows Registry files in ASCII format and output the results in an XML-based format

Usage

Example: 

 require 'wired-diff'
 include Wired::Diff
 ...
 meta = WinRegDiffMetadata.new(baseline.value, delta.value, app_name.value, nsrl.value, action.value)
 ...
 diff = WinRegDiff.new(fp_baseline, fp_delta)
 ...
 doc = WinRegDiffDoc.new(meta, diff)
 doc.write(fp_output)
