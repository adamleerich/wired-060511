Class CmdLn::InputFileNameParameter
In: cmdln.rb
Parent: Parameter

Implementation of a parameter designating an input file. Tests to ensure than the file referred to exists, is a normal file and is readable. Throws CmdLn::FileArgumentError if the test fails. This exception is handled by CmdLn::Parameters#process_parameters
